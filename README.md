# Thor's Doors Vercel Site

This folder is ready to upload to Vercel as a static site.

Files included:
- index.html
- door-before.png
- door-after.png
- materials.png
- thor-logo-white.jpg
- vercel.json

Deploy options:
1. Drag this folder into Vercel as a new project, or
2. Push these files to a GitHub repo and import the repo into Vercel.

For a custom URL, use a subdomain like doors.thorcontractors.com and connect it inside Vercel project settings.
